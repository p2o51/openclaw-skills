# Get 笔记视频总结

使用 Get 笔记网站将视频链接自动转为带 AI 总结的笔记。

## 支持平台

- B站 (bilibili.com / b23.tv)
- 小红书 (xiaohongshu.com / xhsm.com)
- 抖音 (douyin.com)
- YouTube (youtube.com)
- 其他主流视频平台

## 操作流程

### 1. 打开 Get 笔记网站

```
https://www.biji.com/note
```

### 2. 登录账号

- 点击"注册/登录"
- 使用手机验证码登录

### 3. 添加视频链接

在笔记页面找到 **"添加链接"** 按钮（图标为链接形状），点击后输入视频链接，然后点击"生成笔记"。

### 4. 等待 AI 生成

AI 会自动：
1. 抓取视频
2. 提取字幕/音频转写
3. 生成结构化总结报告

### 5. 查看结果

生成完成后，会生成两个页面：

| 页面 | 链接格式 | 内容 |
|------|----------|------|
| **笔记主页** | `/note/{id}` | AI 总结的精华版，包含结构化分析、表格、重点归纳 |
| **全文转写** | `/note/{id}/web` | 完整的视频逐字转录稿，一段一段还原视频内容 |

根据需求选择查看：
- 想看重点归纳 → 访问笔记主页
- 想看完整内容 → 访问全文转写

## 链接格式

| 平台 | 示例 |
|------|------|
| B站短链 | `https://b23.tv/xxxxxx` |
| B站长链 | `https://www.bilibili.com/video/BVxxxxxx` |
| 小红书 | `https://www.xiaohongshu.com/explore/xxxxxx` |
| 抖音 | `https://www.douyin.com/video/xxxxxx` |
| YouTube | `https://www.youtube.com/watch?v=xxxxxx` |

## 速度与精度

- **生成时间**: 约 30-60 秒（视频时长 10-15 分钟）
- **总结质量**: AI 生成结构化报告，包含背景、观点、对比分析
- **转录完整度**: 逐字稿几乎 100% 还原视频内容

## 注意事项

1. 需要登录账号才能使用
2. 部分视频可能因平台限制无法抓取
3. 生成的笔记会自动保存到用户账号
4. 笔记支持追加编辑、分享、标签管理
